/**
 * 选择排序：选出最小的元素
 *     1，找到数组中最小的那个元素，将它与数组的第一个元素交换（如果第一个元素就是最小的，与自己进行交换）
 *     2，在剩下的数组元素中找到最小的，与第二个元素交换位置。
 *     3，循环上面的操作
 * 
 *  （1）运行时间与输入排序状态无关：扫描一遍数组，并不能为下一遍扫描提供什么信息。即使是排好的数组，也和随机排列的数组所需时间一样长。
 *  （2）数据移动最小，为N，与数组长度呈线性关系。（大多数增长的数量级都是线性对数或平方）
 *  
 */
	
	public static void sort(Comparable[] a) {
		int N = a.length;
		for(int i=0; i<N; i++) {
			int min = i;
			for(int j=i+1; j<N; j++) {
				if(less(a[j], a[min])) min=j;
			exch(a, i, min);
			}
		}
	}
